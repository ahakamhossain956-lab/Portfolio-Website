# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML5** and **CSS3**. This project follows a **mobile-first** approach, ensuring a seamless experience across mobile, tablet, laptop, and desktop devices.

## 🚀 Features

- 📱 Mobile-first responsive design
- 💻 Fully responsive for all screen sizes
- 🎨 Clean and modern user interface
- ⚡ Lightweight and fast loading
- 🔤 Google Fonts integration
- ⭐ Font Awesome icons
- 🌍 Cross-browser compatible

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts
- Font Awesome

## 📱 Responsive Design

This website was designed using the **mobile-first** methodology and then enhanced for larger screens.

Supported devices:

- 📱 Mobile
- 📟 Tablet
- 💻 Laptop
- 🖥️ Desktop
